Hogyan távolítsuk el a PowerShell parancsértelmezőt a számítógépről?

Az önállóan telepített PowerShell programnak egyszerre akár több verziója is telepítve lehet a számítógépen, és előfordulhat, hogy a régebbi verziókra már nincs szükség. Ezeket az adott platformra jellemző módon távolíthatjuk el a számítógépről, Windows operációs rendszeren például a Gépházból elérhető Alkalmazások és szolgáltatások beállításlapon, vagy a Vezérlőpultról elérhető Programok és szolgáltatások beállításablakban.

A PowerShell globális eszköz eltávolítása

A .NET globális eszközként telepített PowerShell programcsomag valamennyi platformon egységesen a dotnet tool uninstall --global PowerShell utasítással távolítható el a számítógépről.
